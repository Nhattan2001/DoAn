# KhoaLuan_CNTP_10DHTH

## Hướng dẫn cài đặt

### Cài đặt môi trường

```
python -m venv .venv
```

### Kích hoạt môi trường

- Windows

```
.venv\Scripts\activate
```

- Linux

``` 
source .venv/bin/activate
```

### Cài đặt các thư viện cần thiết

```
pip install -r requirements.txt
```

### Chạy chương trình

```
python main.py
```
